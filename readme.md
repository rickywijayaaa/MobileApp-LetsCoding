
# MobileApp-LetsCoding

MobileApp-LetsCoding adalah aplikasi mobile yang dirancang untuk mempermudah proses pembelajaran coding. Aplikasi ini mendukung lintas platform (Android, iOS, dan Web) dengan memanfaatkan teknologi **React Native** dan **Expo**.

---

## 📥 Instalasi

Ikuti langkah-langkah berikut untuk menjalankan aplikasi di perangkat Anda:

### 1. Clone Repository
Clone repository ini menggunakan perintah berikut:
```bash
git clone https://github.com/rickywijayaaa/MobileApp-LetsCoding.git
```

### 2. Masuk ke Direktori Proyek
Pindah ke direktori hasil clone:
```bash
cd MobileApp-LetsCoding
```

### 3. Install Dependencies
Instal semua dependensi yang diperlukan dengan perintah berikut:
```bash
npm --force install
```

### 4. Jalankan Aplikasi
Jalankan aplikasi menggunakan **Expo CLI**:
```bash
npx expo start
```

Expo akan membuka browser untuk memilih platform target. Anda dapat menjalankan aplikasi di:
- **Perangkat Android/iOS** menggunakan aplikasi **Expo Go**.
- **Browser** untuk mode **web**.

---

## 🌟 Fitur Utama
1. **Kursus Interaktif**
   - Materi pembelajaran coding terstruktur dan mudah diikuti.
2. **Quiz dan Latihan**
   - Menguji kemampuan pengguna melalui quiz dan latihan praktikum.
3. **Dukungan Multiplatform**
   - Mendukung perangkat Android, iOS, dan Web.
4. **Desain Responsif**
   - Menyesuaikan tampilan untuk pengalaman pengguna yang optimal di berbagai perangkat.

## 📂 Struktur Proyek
Struktur folder utama proyek:
```
public/
src/
  ├── assets/         # Gambar dan ikon
  ├── components/     # Komponen UI
  ├── data/           # Mock data
  ├── navigation/     # Navigasi aplikasi
  ├── screens/        # Halaman aplikasi
  ├── services/       # Konfigurasi backend dan API
  ├── store/          # Manajemen state
  ├── theme/          # Tema dan styling
  ├── types/          # Tipe data
  ├── utils/          # Utility functions
```

## 💻 Persyaratan Sistem
- **Node.js**: Versi 14 atau lebih baru.
- **npm**: Versi 6 atau lebih baru.
- **Expo CLI**: Pastikan versi terbaru terinstal:
  ```bash
  npm install -g expo-cli
  ```
- **Expo Go**: Aplikasi untuk menjalankan proyek di perangkat Android/iOS.
